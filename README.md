## Get started

1. Get your API key from https://newsapi.org and put it in a .env file with the name EXPO_PUBLIC_API_KEY

2. Install dependencies

   ```bash
   npm install
   ```

3. Start the app

   ```bash
    npx expo start
   ```

## Features

- View top headlines
- Search news
- Save articles

## TECH STACK USED

- React Native Expo
- Tailwind CSS
- React Native Async Storage
- Expo Secure Store
